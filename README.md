# 个人记账管理系统 python1229

## 项目概述

个人记账管理系统是一款基于 Python Django 框架开发的财务记录工具，旨在帮助用户高效管理个人财务，包括收入与支出的记录、管理和统计。

## 技术栈

- 后端：Python Django
- 数据库：MySQL
- 前端：HTML

## 功能模块

- **用户认证**
  - 用户登录
- **个人中心**
  - 个人信息修改
- **记账管理**
  - 收入记录
  - 支出记录
  - 收入记账管理
  - 支出记账管理
- **统计分析**
  - 收入统计
  - 支出统计

## 系统角色

- 普通用户：可以进行登录、修改个人信息、记录和管理财务收支、查看财务统计。

## 运行环境

- Python 环境配置
- Django 框架安装
- MySQL 数据库服务

## 部署步骤

1. 安装 Python 环境和 Django 框架
2. 安装并配置 MySQL 数据库
3. 导入数据库文件
4. 运行 Django 项目


- 简洁的界面设计，易于操作
- 强大的收入支出统计功能，便于用户了解财务状况
